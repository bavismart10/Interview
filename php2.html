PHP Interview Questions

Security

1. What is SQL Injection?
SQL Injection is a security vulnerability where an attacker injects malicious SQL code into database queries.
2. How to prevent SQL Injection?
Use Prepared Statements (PDO/MySQLi), validate inputs, avoid direct query insertion, and limit DB privileges.
3. What is XSS?
Cross-Site Scripting (XSS) is an attack where malicious scripts are injected into web pages.
4. How to prevent XSS?
Use htmlspecialchars(), validate inputs, and enable Content Security Policy (CSP).

OOPS in PHP

5. What is OOPS?
OOPS (Object-Oriented Programming System) is a programming approach where we write code using classes and objects.
6. What are the four pillars of OOPS?
Pillar Meaning
Encapsulation Data hiding using access modifiers
Inheritance Parent → Child relationship
Polymorphism Same method, different behavior
Abstraction Hide implementation details
7. What are access modifiers?
public → Accessible everywhere
protected → Class + Child class
private → Same class only
8. What is constructor & destructor?
Constructor is automatically called when object is created.
Destructor is called when object is destroyed.
9. Difference between self and $this?
$this → Refers to current object.
self:: → Refers to current class.

PHP Design Patterns

10. What are Design Patterns?
Design patterns are reusable solutions to common software design problems that help make PHP code more maintainable, scalable, and testable.

Creational Patterns

Singleton → Ensures only one instance (Example: Database connection)
Factory / Factory Method → Creates objects without exposing logic (Example: Payment gateway)
Builder → Builds complex objects step by step

Structural Patterns

MVC → Separates Model, View, Controller
Adapter → Makes incompatible interfaces work together
Decorator → Adds extra behavior without modifying class
Facade → Provides simple interface to complex system

Behavioral Patterns

Observer → Event listeners, notifications
Strategy → Change algorithm dynamically
Command → Encapsulates actions as objects

Magic Constants

Constant Meaning
__LINE__ Current line number
__FILE__ Full file path
__DIR__ Directory of file
__FUNCTION__ Function name
__CLASS__ Class name
__METHOD__ Method name
__NAMESPACE__ Current namespace

Important PHP Questions (3 Years Experience)

11. Difference between include and require?
include → Warning if file missing.
require → Fatal error if file missing.
12. What is namespace in PHP?
Namespace avoids class name conflicts in large applications.
13. What is Composer?
Composer is a dependency manager for PHP.
14. What is Autoloading?
Automatically loads class files when needed using spl_autoload_register() or Composer.
15. What is Session and Cookie?
Session → Stored on server.
Cookie → Stored in browser.
16. What is REST API in PHP?
REST API allows communication between client and server using HTTP methods.
17. What is PDO?
PDO (PHP Data Objects) is a secure and consistent way to connect to databases.
18. What is Dependency Injection?
Dependency Injection is passing required objects to a class instead of creating inside it.
19. What is Trait in PHP?
Trait allows code reuse in multiple classes.
20. How to improve PHP performance?
Use caching, optimize queries, use indexes, enable OPcache, and avoid unnecessary loops.

Microservices

21. What is Microservices?
Microservices is an architecture where an application is divided into small independent services that communicate via APIs.
