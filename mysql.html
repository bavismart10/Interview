SQL & MySQL Interview Questions

Basic Concepts

1. What is SQL?
SQL (Structured Query Language) is used to store, retrieve, update, and delete data from databases. 
SELECT * FROM users;
2. What is MySQL?
MySQL is an open-source relational database management system used with web applications.
3. Difference between SQL and MySQL?
SQL is a language.
MySQL is a database system that uses SQL.

Queries & Commands

4. What is a Primary Key?
A primary key uniquely identifies each record in a table and does not allow NULL values. 
CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100)
);
5. What is a Foreign Key?
A foreign key creates a relationship between two tables. 
ALTER TABLE orders
ADD CONSTRAINT fk_user
FOREIGN KEY (user_id) REFERENCES users(id);
6. Difference between DELETE, TRUNCATE, DROP?
DELETE → Removes selected rows.
TRUNCATE → Removes all rows.
DROP → Deletes the entire table. 
DELETE FROM users WHERE id = 1;
TRUNCATE TABLE users;
DROP TABLE users;
7. What is WHERE clause?
Filters records based on conditions. 
SELECT * FROM users WHERE status = 1;
8. Difference between WHERE and HAVING?
WHERE filters rows before grouping.
HAVING filters after GROUP BY. 
SELECT department, COUNT(*)
FROM employees
GROUP BY department
HAVING COUNT(*) > 5;
9. What is GROUP BY?
Groups rows with the same values and is used with aggregate functions.
10. What are Aggregate Functions?
COUNT(), SUM(), AVG(), MIN(), MAX() 
SELECT COUNT(*), SUM(salary), AVG(salary)
FROM employees;

Joins

11. What is JOIN?
JOIN combines rows from two or more tables based on a related column. 
SELECT users.name, orders.amount
FROM users
INNER JOIN orders ON users.id = orders.user_id;
12. Types of JOINs?
INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL JOIN.
13. Difference between INNER JOIN and LEFT JOIN?
INNER JOIN → Returns matching records only.
LEFT JOIN → Returns all records from left table. 
SELECT *
FROM users
LEFT JOIN orders ON users.id = orders.user_id;

Indexes & Performance

14. What is an Index?
An index improves query performance by speeding up data retrieval. 
CREATE INDEX idx_name ON users(name);
15. What is Normalization?
Normalization organizes data to reduce redundancy.
16. What is Denormalization?
Denormalization improves read performance by adding redundancy.

Security & Transactions

17. What is SQL Injection?
A security attack where malicious SQL code is injected into queries.
18. How to prevent SQL Injection?
Using prepared statements and parameterized queries. 
$stmt = $conn->prepare("SELECT * FROM users WHERE email = ?");
$stmt->execute([$email]);
19. What is a Transaction?
A transaction is a set of queries executed as a single unit. 
START TRANSACTION;
INSERT INTO orders VALUES (...);
COMMIT;
20. What are ACID Properties?
Atomicity, Consistency, Isolation, Durability.

Advanced / Tricky Questions

21. Difference between UNION and UNION ALL?
UNION removes duplicates.
UNION ALL keeps duplicates and is faster. 
SELECT name FROM users
UNION
SELECT name FROM employees;
22. What is Subquery?
A query inside another query. 
SELECT * FROM users
WHERE id IN (SELECT user_id FROM orders);
23. What is AUTO_INCREMENT?
Automatically generates unique numbers for primary key columns.
24. Difference between CHAR and VARCHAR?
CHAR → Fixed length.
VARCHAR → Variable length.
25. What is EXPLAIN?
Used to analyze and optimize query performance. 
EXPLAIN SELECT * FROM users WHERE name = 'John';
