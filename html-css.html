HTML & CSS Interview Questions

HTML Questions

1. What is HTML?
HTML (HyperText Markup Language) is used to structure web pages.
2. Difference between HTML and HTML5?
HTML5 supports semantic tags, audio/video, local storage, and better mobile support.
3. What are Semantic Tags?
Semantic tags describe the meaning and purpose of the content.
Examples: <header>, <nav>, <main>, <section>, <article>, <footer>, <aside>
4. Difference between div and span?
div → Block-level element.
span → Inline-level element.
5. What is iframe?
An iframe embeds another HTML page within the current page.
6. What are Meta Tags?
Meta tags provide page information like description, charset, and viewport.
Example: <meta name="viewport" content="width=device-width, initial-scale=1.0">
7. What are block-level and inline elements?
Block elements take full width (div, p, h1).
Inline elements take only required width (span, a, strong).
8. What is the difference between GET and POST?
GET → Data sent via URL, less secure.
POST → Data sent in body, more secure.
9. What is localStorage and sessionStorage in HTML5?
localStorage → Stores data permanently.
sessionStorage → Stores data until tab/browser closes.

CSS Questions

10. What is CSS?
CSS styles and layouts web pages by controlling colors, fonts, spacing, and positioning.
11. Difference between class and id in CSS?
Class → Reusable multiple times.
ID → Must be unique.
12. What is the CSS Box Model?

The CSS Box Model describes how elements are structured and spaced on a webpage.

It consists of four parts:

Content → The actual text or image inside the element.

Padding → Space between content and border.

Border → Surrounds the padding and content.

Margin → Space outside the border.

Total width = Content + Padding + Border + Margin

div {
  width: 200px;
  padding: 10px;
  border: 5px solid black;
  margin: 20px;
}
13. What is display property?

The display property defines how an element is displayed on the page.

block → Takes full width and starts on a new line (e.g., div, p).

inline → Takes only required width and stays in the same line (e.g., span, a).

inline-block → Like inline, but width and height can be set.

none → Hides the element (removes it from layout).

flex → Enables Flexbox layout.

grid → Enables Grid layout.

14. What is position property?

The position property defines how an element is positioned on a webpage.

static → Default position (normal document flow).

relative → Positioned relative to itself.

absolute → Positioned relative to nearest positioned parent.

fixed → Positioned relative to viewport (does not scroll).

sticky → Acts like relative until a scroll point, then becomes fixed. (or) When you scroll to a certain height/position, it sticks to the screen like fixed.

15. What is Flexbox?
Flexbox is used to align items in a single row or column. 
.container {
  display: flex;
  justify-content: center;
  align-items: center;
}
16. What is CSS Grid?
Grid is used to create two-dimensional layouts using rows and columns.
17. What is Responsive Design?
Responsive design adapts layout for different screen sizes using media queries.

Common Media Query Breakpoints:

Mobile: max-width: 767px

Tablet: min-width: 768px and max-width: 1023px

Desktop / PC: min-width: 1024px

@media (max-width: 768px) {
  body {
    background: lightgray;
  }
}
18. Difference between visibility: hidden and display: none?
visibility: hidden → Hides element but keeps space.
display: none → Removes element from layout.
19. What is z-index?
Controls stacking order of positioned elements. It decides which element comes in front when elements overlap.
20. What is specificity in CSS?
Specificity determines which CSS rule is applied when multiple rules target the same element. Priority: Inline > ID > Class > Element
21. What is the difference between relative and absolute positioning?
relative → Positioned relative to itself.
absolute → Positioned relative to nearest positioned parent.
22. What is overflow property?
Controls content overflow: visible, hidden, scroll, auto.
23. What is pseudo-class?

A pseudo-class is used to define a special state of an element.

It is written using a single colon (:).

Examples:

:hover → When the mouse is over an element.

:focus → When an input field is focused.

:active → When an input field is being activated , such as being clicked

:first-child → Selects the first child element of its parent.

:nth-child() → Selects the nth child element of its parent.

24. What are pseudo-elements?

A pseudo-element is used to style a specific part of an element.

It is written using a double colon (::).

Examples:

::before → Inserts content before an element.

::after → Inserts content after an element.

::placeholder → Styles the placeholder text within input or textarea form elements.

::first-letter → Styles the first letter of text.

25. What is difference between em and rem?
em → Relative to parent font-size (or) relative to parent element
rem → Relative to root (html) font-size. (or) relative to root html element. 
/* em */
div {
  font-size: 20px;
}
span {
  font-size: 1.5em; /* 30px */
}
span small {
  font-size: 0.5em; /* 0.5 x 30px = 15px */
}

/* rem */
html {
  font-size: 16px; /* Default is 16px */
}
p {
  font-size: 2rem; /* 2 x 18px = 36px */
}
div p {
  font-size: 1rem; /* 1 x 18px = 18px, still relative to root */
}
26. What is the difference between defer and async in script tag?
async → Downloads script in parallel and executes immediately.
defer → Downloads in parallel but executes after HTML parsing is complete.
27. What is Critical Rendering Path?
It is the process browser follows to convert HTML, CSS, and JS into pixels on screen. Optimizing it improves page load performance.
28. What is the difference between cookies, localStorage, and sessionStorage?
Cookies → Stored on the browser and sent to the server with every HTTP request..
localStorage → Stored in browser permanently; not sent to the server.
sessionStorage → Stored in the browser until tab closes; not sent to the server.
29. What is Web Accessibility (a11y)?
Accessibility ensures websites are usable by people with disabilities using screen readers and keyboard navigation.
30. What is ARIA?
ARIA (Accessible Rich Internet Applications) attributes improve accessibility of dynamic content. Example: aria-label, aria-hidden.
31. What is the difference between semantic HTML and non-semantic HTML?
Semantic → <article>, <section> (meaningful).
Non-semantic → <div>, <span> (no meaning).
32. What is lazy loading?
Lazy loading delays loading of images or components until they are needed. Example: <img loading="lazy">
33. What is padding vs margin?

Simple Difference:

Padding adds space inside the box.

Margin adds space outside the box.

Flexbox Properties

1. Flex Container Properties

display
display: flex; → Enables flexbox.
flex-direction
row | row-reverse | column | column-reverse
justify-content
Aligns items horizontally (main axis). flex-start | flex-end | center | space-between | space-around | space-evenly
align-items
Aligns items vertically (cross axis). stretch | flex-start | flex-end | center | baseline
align-content
Aligns multiple rows when there is extra space.
flex-wrap
nowrap | wrap | wrap-reverse
gap
Adds spacing between flex items. gap: 10px;

2. Flex Item Properties

flex-grow
Defines how much item grows relative to others.
flex-shrink
Defines how much item shrinks.
flex-basis
Initial size of flex item.
flex
Shorthand: flex: grow shrink basis; Example: flex: 1 1 200px;
align-self
Overrides align-items for individual item.
order
Changes visual order of items.
Difference between align-items and align-content
align-items → Aligns individual items on the cross-axis inside a single row or column.
align-content → Aligns multiple rows (flex lines) inside the container when there is extra space.
Difference between justify-items and justify-content
justify-items → Aligns individual items horizontally inside their grid cell. (Used mainly in CSS Grid).
justify-content → Aligns the entire row or grid horizontally inside the container when there is extra space.

Grid Properties

1. Grid Container Properties

display
display: grid; → Enables CSS Grid.
grid-template-columns
Defines column structure. Example: grid-template-columns: 1fr 1fr 1fr;
grid-template-rows
Defines row structure.
grid-template
Shorthand for rows and columns.
gap
Sets spacing between rows and columns. gap: 10px;
row-gap / column-gap
Sets row and column spacing separately.
justify-items
Aligns items horizontally inside grid cell.
align-items
Aligns items vertically inside grid cell.
place-items
Shorthand for align-items and justify-items.
justify-content
Aligns entire grid horizontally.
align-content
Aligns entire grid vertically.

2. Grid Item Properties

grid-column
Defines column start and end. Example: grid-column: 1 / 3;
grid-row
Defines row start and end.
grid-area
Shorthand for grid-row and grid-column.
justify-self
Aligns individual item horizontally.
align-self
Aligns individual item vertically.

Advanced CSS Questions (4+ Years)

1. What are vh and vw units?

vh (viewport height) → Based on screen height.

1vh = 1% of total screen height.

Example: height: 100vh; (Takes full screen height)

vw (viewport width) → Based on screen width.

1vw = 1% of total screen width.

Example: width: 100vw; (Takes full screen width)

2. Height vs min-height vs max-height

height → Sets exact height. It will not change.

min-height → Sets minimum height. The height will not go below this value, but it can grow bigger if the content increases.

max-height → Sets maximum height. It will not grow beyond this value.

3. Width vs min-width vs max-width

width → Sets exact width.

min-width → Sets minimum width. The width will not go below this value, but it can grow bigger if the content increases.

max-width → Sets maximum width. It will not grow beyond this value.

4. What is CSS specificity calculation?
Priority Orders Inline (1000 Points) > ID (100 Points) > Class (10 Points) > Element (1 Point).
5. What is BEM methodology?

BEM (Block, Element, Modifier)

Block Element Modifier is a naming convention for writing scalable and maintainable CSS. It is avoids CSS conflicts.

Example: card__title--active 
menu
menu__item
menu__item--active
6. What is the difference between Flexbox and Grid?
Flexbox → One-dimensional layout (row OR column).
Grid → Two-dimensional layout (rows AND columns).
7. What is CSS preprocessor?
Tools like SASS or LESS that add variables, nesting, and functions to CSS.
8. What is CSS variable?
Custom properties defined using --variable-name. Example: :root { --main-color: blue; }
9. What is repaint and reflow?
Reflow → Layout recalculation when DOM changes.
Repaint → Visual update without layout change.
10. What is will-change property?
It hints the browser about upcoming changes to optimize performance.
11. What is media query best practice?
Use mobile-first approach. Example: @media (min-width: 768px) { }
12. What is object-fit property?
Controls how image fits inside container (cover, contain).
13. What is clamp() in CSS?
clamp() sets responsive size with minimum, preferred, and maximum values.

It takes three values:

clamp(min, preferred, max)

Example:
h1 {
  font-size: clamp(16px, 5vw, 40px);
}
14. What is CSS animation vs transition?
Transition → Used for simple changes between two states (start and end).
Animation → For complex keyframe animations, It works when a property changes, like hover.
15. What is @keyframes in CSS?

@keyframes is used to define animation steps in CSS.

It specifies how an element should change from start to end.

Example:

@keyframes slide {
  0%   { left: 0; }
  50%  { left: 50px; }
  100% { left: 100px; }
}

.box {
  position: relative;
  animation: slide 2s infinite;
}

This means the element moves from 0px to 100px in 2 seconds and repeats forever.

16. How to optimize CSS performance?
Minify CSS, remove unused styles, avoid deep selectors, use caching, and reduce reflows.
