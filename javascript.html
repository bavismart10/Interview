JavaScript Interview Questions

JavaScript Basics

1. What is JavaScript?
JavaScript is a client-side scripting language used to make web pages interactive and dynamic.
2. Difference between var, let, and const?
var → Function-scoped, can be redeclared.
let → Block-scoped, cannot be redeclared.
const → Block-scoped, cannot be reassigned.
3. What is Hoisting?
Hoisting moves variable and function declarations to the top of their scope before execution.
4. What are Data Types?
Primitive: String, Number, Boolean, Undefined, Null, Symbol, BigInt.
Non-Primitive: Object, Array, Function.

Functions & Scope

5. What is a Function?
A reusable block of code that performs a specific task.
6. Arrow Function vs Normal Function?
Arrow functions do not have their own this, normal functions do.
7. What is Closure?
A closure remembers variables from its outer scope even after the outer function finishes execution.

DOM & Events

8. What is DOM?
DOM represents the HTML page as objects that JavaScript can manipulate.
9. How to select elements?
getElementById(), getElementsByClassName(), querySelector(), querySelectorAll()
10. What is Event Bubbling?
Event bubbling means an event moves from child element to parent element.
11. Difference between == and ===?
== compares only value.
=== compares value and data type.

Asynchronous JavaScript

12. What is Asynchronous JavaScript?
It allows tasks like API calls to run in the background without blocking the main thread.
13. What is a Callback?
A function passed as an argument and executed later.
14. What is a Promise?
An object representing completion or failure of an async operation.
States: Pending, Fulfilled, Rejected.
15. What are async and await?
Keywords used to handle promises in a cleaner synchronous-looking way.

Arrays & Objects

16. Common Array Methods?
map(), filter(), reduce(), forEach(), push(), pop()
17. Difference between null and undefined?
undefined → Declared but not assigned.
null → Intentionally empty.
18. What is Destructuring?
Extract values from arrays or objects into variables easily. 
const user = { name: "Bavi", age: 25 };
const { name, age } = user;

const arr = [10, 20, 30];
const [a, b, c] = arr;

ES6+ & Browser

19. What is Template Literal?
Uses backticks (`) and allows string interpolation using ${}.
20. What is Spread Operator?
The spread operator (...) expands arrays or objects into elements.
21. localStorage vs sessionStorage?
localStorage → Stores data permanently.
sessionStorage → Stores data until browser closes.
22. What is CORS?
CORS controls how resources are shared between different domains.
23. Is JavaScript synchronous or asynchronous?
JavaScript is single-threaded and synchronous, but supports async operations using callbacks, promises, and async/await.
